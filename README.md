## Instalación
[⬅️Terminos & Condiciones(README.md)
- **Plataforma:** Android
- Descarga el archivo APK desde la sección de [releases](https://github.com/GalenKishi/FurEverHome/releases).
- Asegúrate de habilitar la instalación de aplicaciones de fuentes desconocidas en tu dispositivo Android.
- Ejecuta el APK e instala el juego.
- ¡Comienza a jugar y disfruta de mejoras, eventos, y la experiencia de cuidar tu refugio!

### Requisitos mínimos

- Android 6.0 (Marshmallow) o superior
- 100 MB de espacio libre
- Conexión a Internet recomendada para eventos y actualizaciones

### Soporte

Si tienes problemas con la instalación o el funcionamiento del juego, consulta la sección de issues o abre una nueva incidencia describiendo tu problema.
